# Roadmap

PIXTEL is under active development. Here's what's been built and what's coming next.

## Completed

### Core Platform
- [x] Browser-based WebGL game (no download required)
- [x] Guest and Phantom wallet authentication
- [x] Character selection (27 unique pixel-art avatars)
- [x] Customizable nicknames

### Social Features
- [x] Real-time room chat
- [x] Friends system (add, accept, reject, remove)
- [x] Direct messaging (DMs)
- [x] Online presence tracking
- [x] Player location sharing (see which room friends are in)

### Minigames
- [x] Dice — 2-4 player betting minigame
- [x] 8-Ball Pool — 1v1 with real-time physics
- [x] Server-authoritative minigame logic (anti-cheat)
- [x] XIPS betting system (50K - 10M range)
- [x] Automatic pot distribution
- [x] Pool cue customization

### Rooms & Hotel
- [x] Public community rooms (Bar, Casino, Gym, Disco, Lobby, Street, Classroom)
- [x] Player-created rooms (public/private)
- [x] Room decoration with furniture
- [x] Grid-based placement system
- [x] Room size options (Small, Medium, Large)

### Economy
- [x] XIPS in-game currency
- [x] Shop with furniture and cues (paid with PIXTEL token)
- [x] Inventory system
- [x] Starter items for new players

### Crypto
- [x] Phantom wallet integration
- [x] PIXTEL token holder verification
- [x] Fireplace token burning (deflationary mechanism)
- [x] On-chain transaction verification
